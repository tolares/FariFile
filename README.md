## Structure du dépôt

## Gestion de projet

Un compte rendu de projet est disponible dans `report/report.pdf`.

Les comptes rendus de réunions sont disponibles dans `report/crr*.pdf`

## Utilisation

```sh
make
chmod +x fari
cp fari $PROJECT
cd $PROJECT
./fari $FARIFILE
```
